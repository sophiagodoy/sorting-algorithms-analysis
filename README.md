# Sorting Algorithms Analysis

This project presents an experimental analysis of sorting algorithms.

## Algorithms Compared
- Selection Sort
- Merge Sort
- Heap Sort

## Input Sizes
- 1,000 elements
- 10,000 elements
- 100,000 elements

## Metrics Evaluated
- Execution time
- Average execution time
- Standard deviation
- Number of swaps/movements

## Development Environment
- Python
- Google Colab

## Objective
Compare the practical performance of different sorting algorithms and verify whether the experimental results are consistent with their theoretical time complexities.
